## Farmland

Une application python initialement crée pour l'automatisation des récoltes sur le
mini jeux Discord "Farmland" mais ensuite upgrader en une application d'automatisation plus globale

## Installation:

### Prérequis:

- Python >= 3.14

Cloner le dépot, installer les dépendance avec ``pip install -r .\requirements.txt``
et lancer ``gui.py``
